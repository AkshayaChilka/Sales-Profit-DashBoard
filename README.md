📊 Sales and Profit Dashboard (Microsoft Excel)
📌 Project Overview

The Sales and Profit Dashboard is my first interactive dashboard built entirely in Microsoft Excel. This project was created to analyze sales performance, profit trends, customer insights, and regional sales using a single, easy-to-understand dashboard.

Building this dashboard took time because I was learning many Excel features while developing it. It was a great hands-on experience that helped me understand how to transform raw data into meaningful business insights.

🎯 Problem Statement

Businesses often have large amounts of sales data spread across multiple sheets, making it difficult to identify trends and make informed decisions.

This dashboard solves that problem by providing a centralized view of key business metrics through interactive visualizations.

🎯 Objectives
Analyze total sales and total profit.
Identify top-performing product categories.
Compare yearly profit trends.
Track monthly sales performance.
Find top customers based on profit.
Analyze customer distribution by year.
Visualize sales across different states.
Enable interactive filtering using slicers.
